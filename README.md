# pro·em

> –noun
> an introductory discourse; introduction; preface; preamble.

> Origin:
> 1350–1400; < L prooemium < Gk prooímion prelude (pro- pro-2 + oím(ē) song + -ion dim. suffix);
> r. ME proheme < MF < L, as above
