Proem by trq

A small, simple, quick PHP MVC

Proem is currently undergoing a complete rethink / redesign and the code currently within the repository is not likely to resemble the end framework at all.

The main motivation for the rework is to take advantage of some of the new features provided by PHP5.4 which is still in the release candidate stage.

You can clone the project with Git by running: 

$ git clone git://github.com/trq/proem

get the source code on GitHub : trq/proem
